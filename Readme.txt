Projekti kirjeldus:

Projekt, 4 vaheulesannet
On valine API, kuskohast saab parida linnade ilma infot:
Hetke ilm ja lisaks 3 paeva ilmaennustus (API valjundis antakse ennustus iga kolme tunni tagant)

Vaja on leida:
* hetke temperatuur
* 3me paeva ilmaennustusest leida iga paeva korgeim ja madalaim temperatuur 
* Koordinaadid (GEO) kujul xxx:yyy

API valjund on JSON formaadis
Kood esitada Clean Code nouetele vastavalt
Kood laadida tahtajaks GitHubi, kusjuures iga vahetulemus peab olema eraldi valja toodav (diffitav)
